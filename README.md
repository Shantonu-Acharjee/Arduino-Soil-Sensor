# Arduino-Soil-Sensor

## SD Card
Arduino --> Master
SD Card Module ---> Slave

SCK - Serial Clock
MISO - Master In Slave Out (Card will send data to Arduino)
MOSI - Master Out Slave In (Arduino Will Send Data to Card)
SS - Slave Select (if high communication off | if low communication on)
